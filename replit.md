# My Make File - React + Vite Project

## Overview
A React frontend application built with Vite, Tailwind CSS, and various UI component libraries (Radix UI, MUI, shadcn-style components).

## Project Structure
- `/src/` - Source code
  - `/app/` - Main application components
  - `/styles/` - CSS styles
  - `main.tsx` - Application entry point
- `index.html` - HTML entry point
- `vite.config.ts` - Vite configuration

## Tech Stack
- React 18
- Vite 6.3.5
- Tailwind CSS 4
- Radix UI components
- MUI (Material-UI)
- Various utility libraries (date-fns, recharts, react-hook-form, etc.)

## Development
- **Port**: 5000
- **Start**: `npm run dev`
- **Build**: `npm run build`
